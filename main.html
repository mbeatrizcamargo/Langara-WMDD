About this project

This website was created to gather all my notes and projects related to my WMDD course at Langara College.

The notes are divided by term, subject and week. Each week's notes page was made using only content from that specific week's lecture and material. So the first weeks may seem very simple, but the design and code will improve as you move forward.

Each new week will add knowledge to the last one, and will combine concepts from different subjects.

Basic structures and styles have been used on all pages, to improve readability and keep all the layouts cohesive.
